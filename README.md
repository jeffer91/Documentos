# Proyectos IA

App inteligente para gestionar proyectos con enfoque en aumentar ganancias.

La app no busca ser solo una lista de tareas. Su objetivo principal es ayudar a decidir qué proyecto avanzar, medir rentabilidad, calcular utilidad, registrar horas trabajadas, analizar avance real y recomendar acciones concretas con apoyo de IA.

---

## Objetivo principal

Responder cada día:

```text
¿Qué proyecto me conviene avanzar hoy para ganar más dinero o acercarme más a generar dinero?
```

---

## Ideas clave

- Gestionar proyectos de diferentes tipos.
- Medir avance, horas, ingresos, gastos y utilidad.
- Calcular dinero por hora.
- Calcular punto de equilibrio según el tipo de proyecto.
- Detectar proyectos atrasados, débiles o sin modelo claro.
- Recomendar el Top 3 de proyectos para avanzar cada día.
- Usar IA para diagnosticar, resumir, priorizar y corregir decisiones.
- Mantener una interfaz minimalista y fácil de usar.
